# The Range

Your blood is chemically different than it was in 1999.

Not because you changed. Because the atmosphere changed.

Researchers in Australia found that serum bicarbonate levels in the US population have risen 7% since 1999, tracking increases in atmospheric CO₂. Your body absorbs carbon dioxide. Your blood buffers it. The buffer shifts your chemistry. Measurably. Across the entire population.

Every individual measurement falls within the "normal range."

The normal range has not been updated.

---

This is the problem with a ruler that moves with what it measures.

A blood test in 2026 returns a number. The lab compares it to a reference range. The range was defined using population data. The population changed. The range should have moved. It did not.

So a result that would have been flagged in 1999 passes in 2026. Not because it is healthier. Because the definition of healthy drifted.

No alarm fires. Not because nothing changed. Because the alarm's calibration moved with the thing it was calibrated to detect.

---

This pattern is everywhere.

A performance review uses last quarter's numbers as the baseline. Last quarter was already declining. The review says "stable." Stable relative to what?

A model is evaluated against human preferences. Human preferences were shaped by previous model outputs. The benchmark co-evolves with the system it benchmarks. Goodhart's Law usually describes gaming a metric. This is subtler — the metric gaming itself, without anyone trying.

A relationship checks in: "Are we okay?" The answer is compared to recent history. Recent history was already drifting. "Okay" keeps meaning less, so slowly that each check passes.

---

The fix is not better measurement. The fix is anchoring the reference.

An absolute standard. A number from before the drift started. A memory of what "okay" used to mean — not what it has come to mean.

The problem is that anchoring feels paranoid. Everything looks fine. The numbers are in range. Why go back to 1999?

Because the range moved. And it moved so slowly that every step looked like standing still.
